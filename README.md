An AI-powered retail assistant with emotional intelligence, pattern detection, and smart escalation capabilities.

## Features

- **Natural Conversation**: Voice-based interaction with customers
- **Visual Awareness**: Camera-based perception and recognition
- **Expressive Movements**: Engaging head, body, and antenna animations
- **Pattern Detection**: Tracks interaction signals to identify trends
- **Smart Escalation**: Automatically alerts management via Slack for high-priority situations

## Custom Tools

- `record_interaction_signal`: Logs customer interactions with intent, sentiment, and resolution status
- `get_signal_aggregates`: Analyzes patterns across multiple interactions
- `escalate_to_slack`: Sends structured alerts to management team
